Verbos (ou métodos) HTTP

## 1.Get 
  Método usado para ler/conseguir informações/recursos 

## 2.Post
  Método usado para registrar novas informações

## 3.Put
  Método usado para alterar informações

## 4.Delete
  Método usado para apagar informações
